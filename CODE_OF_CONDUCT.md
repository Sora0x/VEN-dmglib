# Code of Conduct

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct.

## In short

We want this library to be a place where contributors of any skill level, background, or identity can participate without friction. That means:

- **Be kind.** Direct technical feedback is welcome; personal attacks are not.
- **Assume good intent.** A confused PR or a missed style rule is not a moral failing — it is an opportunity to teach.
- **Stay on topic.** Discussions in issues, PRs, and Discord stay focused on the library's code, design, and use.
- **Respect others' time.** Reviews are best effort; do not pile pressure on volunteers.

For the full list of expected and prohibited behaviors, see the [official Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) text.

## Scope

This Code of Conduct applies to all spaces where the project is represented:

- The GitHub repository (issues, PRs, discussions, code review)
- Discord conversations directly about this project
- Any other forum where someone speaks on behalf of, or as a contributor to, this project

## Reporting

If you experience or witness behavior that goes against this Code of Conduct, contact the maintainer privately:

- **Discord:** `nk_sora` (DM)

Reports are handled confidentially. The maintainer will respond as quickly as possible.

## Enforcement

The maintainer may take any action they deem appropriate, including:

- A private warning
- A request to change behavior
- Temporary or permanent removal from project spaces (issue/PR commenting, repository contributions, Discord channels)

Decisions are at the maintainer's discretion. While the project has a single maintainer, there is no formal appeal process; this will be revisited if the team grows.
